javascript 代码规范

代码规范我们应该遵循古老的原则：“能做并不意味着应该做”。

1.全局命名空间污染

总是将代码包裹在一个立即的函数表达式里面，形成一个独立的模块。

不推荐

var x = 10, y = 100; console.log(window.x + ' ' + window.y);

推荐

;(function(window){ 'use strict'; var x = 10, y = 100; console.log(window.x + ' ' + window.y); }(window));
